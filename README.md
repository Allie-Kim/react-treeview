# React Treeview

### Usage
**Install**
````
npm install
````

**Start the application in development mode**
````
npm start
````
Open http://localhost:8080 in your browser.

**When ready, build for production**
````
npm run build
````

### Data Structure
````
[{
	"children": [{
    	"children": [..],
    	"id": "2",
    	"title": "parent-2"}],
    "id": "1",
    "title": "parent-1"
}]
````
